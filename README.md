# Pinout

The RK919/RK918 uses a Huafenda HFD2201KBA microcontroller. Upon further inspection reveals that the HFD2201KBA is a rebranded SONiX SN32F248B ARM-Cortex M0 based microcontroller. You can find the data sheet and pinout as shown below:

https://www.sonix.com.tw/article-en-4336-30356
