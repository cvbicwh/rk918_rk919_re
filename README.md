# Pinout

The RK919/RK918 uses a Huafenda HFD2201KBA microcontroller. Upon further inspection reveals that the HFD2201KBA is a rebranded SONiX SN32F248B ARM-Cortex M0 based microcontroller. You can find the pinout, along with the data sheet in the repository as shown below:
